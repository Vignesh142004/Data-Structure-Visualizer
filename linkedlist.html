About Linked List

A linked list is a linear data structure where elements are not stored at contiguous memory locations. The elements, also called node, are linked using pointers. It has several operations:

Insert: Add an element to the linked list.
Delete: Remove an element from the linked list.
Traverse: Access each element of the linked list.
